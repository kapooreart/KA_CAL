Common fixes
  1. Open this file's API_BASE constant (search for it near the top of the <script> block) and confirm it's your deployed /exec URL, not the placeholder or a /dev URL.
  2. In Apps Script: Deploy → Manage deployments — Access must be set to Anyone.
  3. If you edited Code.gs after deploying, you must Deploy → Manage deployments → Edit → New version.
  4. Code.gs / SheetSetup.gs must be in the Apps Script project bound to your Sheet (opened via that Sheet's Extensions → Apps Script).
  5. Run setupAllSheets at least once so all tabs exist with headers + seed data.
